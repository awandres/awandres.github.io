Marquee

Marquee is a concert sharing app where users can search the Ticketmaster API for upcoming concerts, form groups with other users, and create shared lists where member can vote on which concerts to attend

Technologies Used:

  • AngularJS
  • Express
  • MongoDB/Mongoose
  • Node.js

Randsom

Randsom is an app that takes a phrase and matches the letters in each word to an image of that letter from the Pinterest API and then scrambles and displays the phrase

  • AngularJS
  • Ruby on Rails

Rails Recipes

In Rails Recipes, users can make make a weekly recipe list, where they can view their schedule, add recipes to certain days of the week, and view a shopping list of all the ingredients they need

  • Ruby on Rails
